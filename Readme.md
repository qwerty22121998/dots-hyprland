# DOTFILE

# Prerequisites
- hyprland: Wayland compositor
- fastfetch: System information display
- swaync: Notification daemon
- rofi: App launcher, dmenu
- waybar: Status bar
- awww: Wallpaper daemon
- cava: Audio visualizer
- wlogout: Logout menu
- grim, slurp: Screenshot tools
- pamixer: Volume control
- kitty: Terminal emulator
- zsh: Shell
- starship: Prompt
- wallust: Color scheme generator
- yazi: TUI file manager
- nwg-look: Display/Workspace manager
- btop: TUI system monitor


# Included scripts
- Reload appearance: CTRL + ALT + R 
- Show keybindings: SUPER + H
- Change hyprland animation: SUPER + SHIFT + A
- Change waybar theme: SUPER + SHIFT + B
- Change wallpaper: SUPER + SHIFT + W
- Screenshot: SUPER + SHIFT + S
